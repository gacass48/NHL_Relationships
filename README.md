# NHL_Relationships

Is there a genetic advantage in professional sports? How can we tell?

We probably can't but there are other interesting trends one can look at. For instance, does having a sibling indicate that you are a better NHL player or not, and which family has produced the best siblings?

This page looks at sibling pairs and compares each one to see who is the most successful. It also compares siblings to non-siblings to see if nepotism plays a factor, or if certain families are genetically predisposed to be better players.

We also compare players with the exact same name, some of whom are related, and some who just had parents who thought the same as others.
